# Tmux

## Install

### Install Tmux

[install](./install.md)

### Install TPM

``` bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

#### Install Plugins

`prefix` + <kbd>I</kbd>

#### Update Plugins

`prefix` + <kbd>U</kbd>

#### Remove Plugins

`prefix` + <kbd>alt</kbd> + <kbd>u</kbd>
- remove/uninstall plugins not on the plugin list


## Configure

[tmux.conf](./tmux.conf)

将配置文件写入到 `~/.tmux.conf`，执行

``` bash
tmux source ~/.tmux.conf
```

加载配置
